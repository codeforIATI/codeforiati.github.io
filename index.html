Tools and guidance contributed by community members, in support of IATI infrastructure, publishers and data users.

Monthly Tech call - a monthly call with 5 minute presentations to share new IATI tech.

iatikit - toolkit for using IATI data. It includes a query language wrapper around XPath, to make dealing with disparate IATI versions easier.

Org ID finder - a tool to help users to find the IDs for IATI organisations.

IATI data dump - a daily snapshot of all IATI data (and metadata) on the IATI Registry, also downloadable via dropbox.

IATI transformer - a tool to transform IATI v1.0x data to the latest version of the IATI standard.

Aid on spreadsheets - a tool that produces IATI data in spreadsheet format targeted at aid recipient governments.

IATI in AIMS - guidance on current approaches to integrating IATI data into aid recipient country systems.
