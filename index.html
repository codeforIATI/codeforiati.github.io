Tools and guidance contributed by community members, in support of IATI infrastructure, publishers and data users.

Monthly Tech call - a monthly call with 5 minute presentations to share new IATI tech.

Datastore query builder - user interface for the new IATI datastore.

iatikit - python toolkit for using IATI data. It includes a query language wrapper around XPath, to make dealing with disparate IATI versions easier.

Org ID finder - find organisation identifiers for IATI organisations.

IATI data dump - a daily snapshot of all IATI data on the IATI Registry.

IATI transformer - transform IATI v1.0x data to the latest version of the IATI standard.

Aid on spreadsheets - produce IATI data in spreadsheet format, targeted at aid recipient governments.

IATI in AIMS - guidance on current approaches to integrating IATI data into aid recipient country systems.
